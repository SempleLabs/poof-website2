# Downloads Folder

This folder contains downloadable resources (PDFs) for the website.

## Required Files

Place the following PDF files in this directory:

1. **bookkeeping-setup-guide.pdf** - Small Business Bookkeeping Setup Guide (24 pages)
2. **expense-categorization-cheat-sheet.pdf** - Expense Categorization Cheat Sheet (2 pages)
3. **monthly-bookkeeping-checklist.pdf** - Monthly Bookkeeping Checklist (4 pages)
4. **year-end-closing-procedures.pdf** - Year-End Closing Procedures (16 pages)

## File Naming Convention

Use lowercase with hyphens for consistency:
- `bookkeeping-setup-guide.pdf`
- `expense-categorization-cheat-sheet.pdf`
- etc.

## Creating PDF Content

Each guide should be professionally designed and include:
- Poof branding (logo, colors)
- Valuable, actionable content
- Clear formatting and structure
- Footer with website URL (poofai.com)

## Tools for PDF Creation

- Canva (templates available)
- Adobe InDesign
- Google Docs (export as PDF)
- Microsoft Word (save as PDF)

## Placeholder Files

Until actual PDFs are ready, you can create placeholder PDFs that say "Coming Soon" to test the download functionality.
